## папки

проект разделен на две папки

- avito_web - мое решение
- avito_back - немного измененный back (настроил CORS для корректной работы: добавил http://localhost:5173 в разрешенные origins)

## Запуск
- бэкенд: cd avito_back && npm start
- фронтенд: cd avito_web && npm run dev (http://localhost:5173)



## Библиотеки

в ходе проекта использовал следующие сторонние библиотеки:

- ant-design - для некоторых компонентов, также брал иконки оттуда
- chart.js - для графиков
- react-redux - для управления состояниями
- tailwind - написание стилей

## доп. фичи

из дополнительных возможностей реализовал:
### Горячие клавиши:
- A - одобрить объявление
- D - отклонить объявление
- → - следующее объявление
- ← - предыдущее объявление

### Продвинутая фильтрация:
Сохранение наборов фильтров

URL-синхронизация фильтров (можно поделиться ссылкой)

### Плавный переход между между объявлениями

Анимация появления карточек


## Loaders
использовал React Router Loaders для предзагрузки данных страниц
